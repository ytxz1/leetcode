# 3043. 最长公共前缀的长度 - 解题思路

## 题目理解

**什么是前缀？**
- 正整数的前缀是其最左边的一位或多位数字
- 例如：`12345` 的前缀有：`1`, `12`, `123`, `1234`, `12345`

**什么是公共前缀？**
- 如果整数 `c` 同时是 `a` 和 `b` 的前缀，那么 `c` 就是 `a` 和 `b` 的公共前缀
- 例如：`5655359` 和 `56554` 的公共前缀有 `5`, `56`, `565`, `5655`

**目标：**
- 从 `arr1` 选一个数 `x`，从 `arr2` 选一个数 `y`
- 找到所有 `(x, y)` 组合中最长公共前缀的长度

---

## 解法一：Trie（前缀树）

### 核心思想

把 `arr1` 中所有数字的每一位当作 Trie 的节点插入，然后用 `arr2` 中的数字在 Trie 中逐位匹配。

### 图解过程

以 `arr1 = [1, 10, 100]`, `arr2 = [1000]` 为例：

**Step 1: 将 arr1 插入 Trie**

```
arr1 中的数字转为字符串：
  1   → "1"
  10  → "10"
  100 → "100"

构建 Trie：
        root
         |
         1          ← '1' 是 "1" 的终点
         |
         0          ← '0' 是 "10" 的终点
         |
         0          ← '0' 是 "100" 的终点
```

**Step 2: 用 arr2 在 Trie 中查找**

```
arr2 中的数字转为字符串：
  1000 → "1000"

从 root 开始逐位匹配：
  '1' → 存在，len=1
  '0' → 存在，len=2
  '0' → 存在，len=3
  '0' → 不存在，停止

最长公共前缀长度 = 3
```

### 代码详解

```cpp
struct TrieNode {
    TrieNode* children[10] = {nullptr};  // 每个节点有 10 个子节点（数字 0-9）
};

class Solution {
public:
    int longestCommonPrefix(vector<int>& arr1, vector<int>& arr2) {
        TrieNode* root = new TrieNode();

        // ===== 第一步：构建 Trie =====
        for (int num : arr1) {
            string s = to_string(num);  // 数字转字符串
            TrieNode* node = root;
            for (char c : s) {
                int idx = c - '0';  // 字符转数字索引
                if (!node->children[idx]) {
                    node->children[idx] = new TrieNode();  // 不存在则创建
                }
                node = node->children[idx];  // 移动到子节点
            }
        }

        // ===== 第二步：查找最长前缀 =====
        int maxLen = 0;
        for (int num : arr2) {
            string s = to_string(num);
            TrieNode* node = root;
            int len = 0;
            for (char c : s) {
                int idx = c - '0';
                if (!node->children[idx]) break;  // 无法继续匹配
                node = node->children[idx];
                len++;  // 匹配成功，长度+1
            }
            maxLen = max(maxLen, len);
        }

        return maxLen;
    }
};
```

### 复杂度分析

- **时间复杂度**：O(N × L1 + M × L2)
  - N = arr1 长度，L1 = arr1 中数字的平均位数
  - M = arr2 长度，L2 = arr2 中数字的平均位数
- **空间复杂度**：O(N × L1)  （Trie 的节点数）

## 解法二：哈希集合（更简洁）

### 核心思想

把 `arr1` 中每个数字的所有前缀都存入一个集合，然后检查 `arr2` 中每个数字的前缀是否存在于集合中。

### 图解过程

以 `arr1 = [123, 456]`, `arr2 = [1234, 789]` 为例：

**Step 1: 生成 arr1 所有前缀**

```
123 的前缀: "1", "12", "123"
456 的前缀: "4", "45", "456"

集合 prefixes = {"1", "12", "123", "4", "45", "456"}
```

**Step 2: 检查 arr2 的前缀**

```
1234 的前缀（从长到短检查）:
  "1234" → 不在集合中
  "123"  → 在集合中！len = 3 ✓

789 的前缀:
  "789" → 不在集合中
  "78"  → 不在集合中
  "7"   → 不在集合中

最长公共前缀长度 = 3
```

### 代码详解

```cpp
class Solution2 {
public:
    int longestCommonPrefix(vector<int>& arr1, vector<int>& arr2) {
        unordered_set<string> prefixes;

        // ===== 第一步：生成 arr1 所有前缀 =====
        for (int num : arr1) {
            string s = to_string(num);
            for (int i = 1; i <= s.size(); i++) {
                prefixes.insert(s.substr(0, i));  // 取前 i 位
            }
        }

        // ===== 第二步：检查 arr2 的前缀 =====
        int maxLen = 0;
        for (int num : arr2) {
            string s = to_string(num);
            for (int i = s.size(); i >= 1; i--) {  // 从最长前缀开始检查
                if (prefixes.count(s.substr(0, i))) {
                    maxLen = max(maxLen, i);
                    break;  // 找到该数字的最长前缀，跳出
                }
            }
        }

        return maxLen;
    }
};
```

### 复杂度分析

- **时间复杂度**：O(N × L1 + M × L2²)
  - 生成前缀：O(N × L1)
  - 检查前缀：O(M × L2²)（因为 substr 操作本身是 O(L2)）
- **空间复杂度**：O(N × L1)  （存储所有前缀）

## 解法三：枚举前缀 + 哈希表（全部使用 int）

### 核心思想

用整数除法 `num / 10` 不断去掉末位数字，自然从长到短枚举所有前缀，用 `unordered_set<int>` 存储前缀值，完全避免字符串转换。

### 关键技巧

```
num = 12345

num / 10 = 1234  → 去掉末位 5
num / 10 = 123   → 去掉末位 4
num / 10 = 12    → 去掉末位 3
num / 10 = 1     → 去掉末位 2
num / 10 = 0     → 结束

这样就从长到短枚举了所有前缀：12345, 1234, 123, 12, 1
```

### 图解过程

以 `arr1 = [123, 456]`, `arr2 = [1234, 789]` 为例：

**Step 1: 将 arr1 的所有前缀（整数）存入哈希集合**

```
123 → 插入 123, 12, 1
456 → 插入 456, 45, 4

集合 prefixes = {1, 12, 123, 4, 45, 456}
```

**Step 2: 检查 arr2 的前缀**

```
1234 枚举前缀（从长到短）:
  1234 → 不在集合中
  123  → 在集合中！位数 = 3 ✓

789 枚举前缀:
  789 → 不在集合中
  78  → 不在集合中
  7   → 不在集合中

最长公共前缀长度 = 3
```

### 代码详解

```cpp
class Solution3 {
public:
    int longestCommonPrefix(vector<int>& arr1, vector<int>& arr2) {
        unordered_set<int> prefixes;

        // ===== 第一步：枚举 arr1 所有前缀（整数形式） =====
        for (int num : arr1) {
            while (num > 0) {
                prefixes.insert(num);  // 当前值就是一个前缀
                num /= 10;             // 去掉末位，得到更短的前缀
            }
        }

        // ===== 第二步：检查 arr2 的前缀 =====
        int maxLen = 0;
        for (int num : arr2) {
            int temp = num;
            while (temp > 0) {
                if (prefixes.count(temp)) {
                    // 计算当前前缀的位数
                    int digits = 0;
                    int t = temp;
                    while (t > 0) {
                        digits++;
                        t /= 10;
                    }
                    maxLen = max(maxLen, digits);
                    break;  // 已找到该数字的最长匹配前缀
                }
                temp /= 10;  // 去掉末位，尝试更短的前缀
            }
        }

        return maxLen;
    }
};
```

### 为什么用 int 比 string 更好？

| 对比项 | int 方式 | string 方式 |
|--------|----------|-------------|
| 存储开销 | 4 字节/整数 | 每个字符 1 字节 + 字符串开销 |
| 前缀生成 | `num /= 10`，O(1) 操作 | `s.substr(0, i)`，O(i) 操作 |
| 哈希计算 | 整数哈希，极快 | 字符串哈希，逐字符计算 |
| 代码简洁度 | 纯算术运算 | 需要类型转换 |

### 复杂度分析

- **时间复杂度**：O(N × L1 + M × L2)
  - 生成前缀：O(N × L1)（每个数字最多 L1 次除法）
  - 检查前缀：O(M × L2)（每个数字最多 L2 次除法 + 查找）
- **空间复杂度**：O(N × L1)（存储所有前缀整数）

## 三种方法对比

| 方法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| Trie | 查找效率高，O(L) 查找 | 需要额外数据结构，代码较复杂 | 数字较多、位数较长 |
| 哈希集合(string) | 代码简洁，易理解 | substr 开销大，字符串转换慢 | 数字较少、快速实现 |
| 哈希表(int) | 全整数运算，效率最高 | 需要额外计算位数 | 通用场景，推荐使用 |

**推荐：** 方法三（枚举前缀 + 哈希表）效率最优，代码简洁，推荐使用。
